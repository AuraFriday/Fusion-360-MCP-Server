## üé¨ Screenshot Guide (Quick Reference)

**What to capture:**

1. **AI Creates a Sketch** - TEXT COMMANDS with AI conversation
2. **Python Execution** - Python code running in Fusion
3. **Generic API Access** - JSON command and response
4. **Context Management** - Multiple commands with $variables
5. **Enhanced Error Reporting** - Detailed error with hints
6. **3D Text Result** - "Fusion Rocks!" in viewport
7. **MCP Tool Ecosystem** - List of available tools

**How to capture:**

- Windows: Win+Shift+S (Snipping Tool)
- macOS: Cmd+Shift+4 (Screenshot)
- Save as PNG
- 1920x1080 resolution minimum

## üö® Common Questions

**Q: Do I really need all 7 screenshots?**
A: Minimum 3, but 7 is recommended. More screenshots = better conversion.

**Q: Can I skip the video?**
A: Video is highly recommended but not required. Apps with videos get 3x more downloads.

**Q: What if my screenshots aren't perfect?**
A: Good enough is better than perfect. You can update them later.

**Q: How long does Autodesk review take?**
A: Typically 1-2 weeks. Sometimes faster, sometimes slower.

**Q: Can I update after submission?**
A: Yes! You can update screenshots, description, etc. anytime.
